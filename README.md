# Aplicativo de Aluguéis de Patinetes
Esse é um projeto de um aplicativo mobile e web para um sistema de aluguéis de patinetes elétricas.

## Gerenciamento de estado com React Query (TanStack Query)

O projeto contém uma configuração mínima do React Query para gerenciamento de estado e cache.

O provider está configurado em `app/providers/ReactQueryProvider.tsx` e usado em `app/_layout.tsx`.
